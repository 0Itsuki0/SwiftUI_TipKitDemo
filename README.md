# SwiftUI: TipKit Demo

A Demo of using TipKit to display Tip. 


Specifically, this demo includes:

- Basic inline tip and popover tip
- Control the displayed timing of the tip with
    - Parameters based rules
    - Event based rules
    - Option
- Monitor and control the presenting state
- Reset Eligibility for a specific tip
- UI/UX customization
    - Tip arrow: on specific edge or pointing to an anchor view
    - Background style
    - Image size and style
    - Entire tip view by creating custom `TipViewStyle`
    - Enable background interaction
